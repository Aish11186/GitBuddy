myRepoMate is your personalised repo onboarding and contribution guideline tool. 
Designed to help beginners break into open source. 
Please refer to requirements.txt.
Clone the repo to run locally



To run this project:

**Platform:** 
Any Chromium-based web browser (Google Chrome, Brave, Microsoft Edge, etc.).
**Setup:**
Go to chrome://extensions/ in your browser.
Enable Developer mode (top right toggle).
Click Load unpacked (top left button) and select the myRepoMate folder.
